# PPO-LSTM Multi-UAV Exploration and Coverage

This repository contains a MATLAB implementation of a multi-UAV exploration framework.

## Included Files
- GridWorldTSP60.m : Multi-agent grid environment
- a_star.m : A* path planner
- resetMap.m : Environment reset helper
- TSP.mlx : Experiment notebook
- TSPLSTM.mlx : PPO-LSTM experiment notebook

## Requirements
- MATLAB
- Reinforcement Learning Toolbox
- Deep Learning Toolbox
- Simulink
